# ED_Lista01.05
## Soma de Sequência

Este desafio consiste em criar uma aplicação, em Java, que receba um número e apresente a somatória da seguinte sequência.
```
                                                S = 1 + 1/2 + 1/3 + 1/4 +...+ 1/N
```

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A soma deve ser realizada por uma função recursiva;
3. Declarar, via comentário, a condição de parada da função.
```
